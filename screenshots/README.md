## Features Demonstrated

### Live Dashboard
Real-time factory monitoring with:
- System performance metrics (CPU, memory, network)
- Zone health distribution (ISA-95 levels)
- Active issues and alerts
- Asset status tracking
- Uptime percentage

### Asset Explorer
Comprehensive asset management with:
- Interactive graph visualization
- Table view with sorting/filtering
- Card grid view
- Team ownership information
- Multi-criteria search

### RCA Analysis
Root cause analysis featuring:
- Multi-state support (offline, error, failed, unreachable, degraded, warning)
- Detailed analysis with thought process and evidence
- Upstream impact tracking (3-hop graph traversal)
- Team contacts and escalation paths
- Status-specific recommendations

### GitOps Drift Detection
5-field drift detection system:
- Status drift (critical priority)
- IP Address drift (high priority)
- Version drift (high priority)
- Config Checksum drift (medium priority)
- Security Zone drift (high priority)
- GitHub integration with real repo paths
- Automated remediation actions

### AI Assistant
Natural language query interface:
- Context-aware responses
- Quick query panel
- Team information lookup
- Asset discovery
